21 Grams

We saw 21 Grams for the $5 Monday movie at Fall Creek Theater today. This is a movie in the genre best described by:

  1. People (men, mostly) screw up
  2. Bad things happen
  3. Some mildly redeeming event occurs
If you like that sort of thing (I don't) and you like being extremely depressed (I don't) you might like this movie (I didn't). Otherwise, run away as fast as you can.
