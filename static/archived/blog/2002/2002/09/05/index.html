Ah, the chicken and the google again

I'm reading about how to submit a site to google. You can add a url to their index, but there's one problem:

The ONLY way a site is included in our index is if our crawlers are able to find it by jumping from link to link on the Web. If we have not found your pages, it is because we have not encountered any links on the web pointing to your site.

Suppose there aren't any links to your site from within Google's existing index? Wouldn't the whole point of submitting a site be to increase the number of links to the site? Why does this page exist if not to add a site without any external links pointing to it?
